User Guidelines

  1. Recommended browser : Google Chrome or Mozilla Firefox
  2. Maximum number of characters is 2048
  3. User can give multiple FASTA format DNA sequences (click on the Example button for example) but maximum number of characters should not exceed 2048
  4. Click on the Example button for example
  5. Click on the Reset button to reset
  6. Click on the Submit button to get the prediction results
  7. If a FASTA format is not valid, it will not give any prediction
  8. Case-insensitive : does not distinguish between lower case letter or upper case letter. (E.g. ACGT, acgt, AcGt have the similar meaning)
  9. Must use FASTA format for the DNA sequences
  10. Each FASTA format DNA sequence must start in a new line
  11. For more about FASTA format visit
  12. A FASTA format DNA sequnce is given below (user can copy and paste it in the textarea)
